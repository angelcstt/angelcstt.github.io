使用Shadowsocks作为代理实现PC登录Line的小技巧

发表于 2019-02-18 | 分类于 实践

Line 作为日本最受欢迎的社交软件，使用率占比高达75%以上，但是迫于服务器被G（伟大的）F（防）W（fire墙）拦截，所以境内无法登录，也使很多想和日本朋友或者同学以及亲人联系的小伙伴们感到及其无奈。

今天就给大家带来在国内用电脑登录Line PC端的小技巧，其实也谈不上技巧，这里暂时只说一下PC端的，如果移动端也有小伙伴需要的话，后续更新。

原理说明

其实很简单，就是使用Line 客户端提供的网络代理功能，使用我们shadowsocks提供的网络代理绕过wall的过程，缺点就是表情及视频无法播放，但是能满足日常的聊天功能，至于延迟问题，就得看你代理服务器了，自己的HK的服务器大概在40-60ms范围内。

0x00 准备工作

  • 首先下载好Line的客户端，点击这里下载或者复制以下链接：

    Mac版本请自行从App store下载，这里不再赘述。

  • 下载代理工具Shadowsocks，windows点击这里或者复制以下链接：

    Mac版本复制下面链接自行下载：

  • 下载完以上工具后，我们的准备工作就基本做完了。当然，前提是你得有一个shadowsocks代理服务器，没有代理的去网上搜一个，shadowsocks服务器的配置及使用这里不再赘述，请善用google。

0x01 Shadowsocks配置

在小飞机的PAC规则中添加Line服务器地址，做相应的规则匹配，让所有访问Line服务器的数据全部走代理，这样就能获取到回程的数据而不被wall拦截。

这里以mac为例，windows同样设置，点击shadowsocks，选择编辑用户PAC规则:

会打开PAC rules 列表：

在PAC规则中添加下列内容：

||line.naver.jp^

添加完毕后保存，这样PAC的自定义规则就完成了，然后我们还需要给端口做一下代理，Line客户端比较213文明，就算你开了ss依旧会走系统公网端口，所以你需要给Line设置代理让其网络访问走你的代理。

首先，在小飞机的设置中查看箭筒的HTTP的端口，点击小飞机的偏好设置：

然后再设置页面中查看ss监听的http端口：

本地地址为 127.0.0.1 端口为1087 // 一般情况下都是这个，除非你做了特殊的更改，记住这两个数据。

然后，打开你的Line客户端，点击右上角的设置按钮：

选择设置进入设置中心：

完成设置后重启Line，然后就可以开始与你的小伙伴愉快地聊天了。

此方法存在的问题：表情和事情无法播放，需要在host中绑定相应的ip，后面有空写文章的话再给大家分享。
