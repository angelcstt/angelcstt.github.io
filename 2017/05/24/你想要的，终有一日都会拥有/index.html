你想要的，终有一日都会拥有

发表于 2017-05-24 | 分类于 随笔

夜深沉。

看似平淡的月色中，我在这里行走，似乎所有的一切都和我没有任何关系。

但是。

真的没有关系么？

楔子

年幼时总喜欢数着手指算日子，期待假期的到来。

可以和小伙伴们肆无忌惮的徜徉在希望的田野上，那充满着慈母印象的黄土地。

在哪里有太多的怀念，有太多的回忆，也有太多的年少时扔给彼此的承诺。

似乎人生从来就不会太顺，不是么？

但是我好像却是非常感激人生的开心与不幸。

大抵这就是历练吧，从什么都没有的地方，到什么都没有的地方。

初遇

以前常听别人说时光总喜欢从指缝间溜走，那时年少，总以为是文人骚客为赋新词强说愁罢了。

而时光静走，当这一天真正来到你的身边，你才知晓，你也即将度过那一段怀念终生的日子了。

似乎从来都没有什么对与错，只要问心无愧，那就坚持往前走，从未有一种心境能让你一往无前。

于是。

在这段日子。

我想用文字来记载一些东西，也许承载的并不多，但也许多年后忆及此处，聊以慰藉罢了。

文思枯竭，竟不知道从哪里写起。

那便就乱七八糟的开始写吧。所有的事情，便都要从解名尽处来说起了。

从2013年9月1日说起。

从迈入这所学校的第一步说起。

在这里，不得不提起一个影响我整个大学生涯甚至影响我整个生活轨迹的人@ZS，不知道你有没有看到这篇文章的一天，但是还是要谢谢你，告诉我如何从0到1的开始。

迈入大学的生活似乎也就这样，在浑浑噩噩中起床，然后又在小电影和万卷手指中昏昏睡去。

而这样的生活，一转眼就是两年，于是，成绩开始挂红灯，红头文件开始传入自己手中。

而从哪里跌倒，就得从哪里爬起来。也就在这时，人生中出现了不可轻失之重@Latin Girl。

对，我想你肯定会看到这里，不要怀疑了，说的就是你。

而相遇，就在那个烈日当空的夏天。

梦想

年幼时就听人常说，人没有梦想和咸鱼无异，要做一只有梦想的咸鱼。

其实我想说的是，咸鱼翻身，依旧是咸鱼。

但是就算是咸鱼，那也是超越。

人生路上，总有太多的不如意，而正是这些不如意构成了命运交响曲的音符和节奏。

所以才有了贝多芬的绝唱。

人终归还是要有梦想的，最起码知道自己想要成为什么样的人。

然后朝这个目标一直坚持走下去，会有坎坷，但是这才叫筑梦啊，要不然整天吹得中国梦岂不是毫无意义了。

而梦想，就是让你知道，这个世界上，还有一些东西是你想要但是还没拿到的。

有一些人是你想结交但是自身资历不够的。

有一些精神是你一直坚持追随的。

大抵，这就是梦想。

年幼时的梦想，简单粗暴，充满童趣和单纯。

而现在，梦想是一盏灯，给你方向，让你坚持。

别让梦想尘封箱底，努力照顾培养，让其熠熠生辉。

延续

时光静走，青葱岁月，弹指间，昨夕稚气懵懂的样子也被时光淘洗的恍若两世。

有人说，大学是梦想着的天堂，是无梦者的温床。

成功与否，也都收拾行李，收拾心情，带走梦想，带走青涩。

从此走上自己想要成为的方向。

而际遇，也从这里开始。

不知觉中有些记忆已经开始倒计时了，容不得我们的任性和不珍惜了。

清风拂晓月，落花掩古城。

是时候说再见了。

而故事，还没有到封笔的时候。

传奇

终有一天。你想要的，你都会拥有。

是的。

只要有想法，然后坚持走向你想要成为的方向。

路在自己脚下。

别人的流言蜚语能耐你何？

坚持，努力。

成功就离你不远。

而你所想所希望的，便是你披星戴月最好的回报。

而你。

就是你自己的传奇。
